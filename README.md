# Extracting and Visualizing Stock Data

This repository contains my submission for the **IBM Data Science** course project:  
**Extracting and Visualizing Stock Data** using Python, `yfinance`, `BeautifulSoup`, and `Plotly`.

## Files
- [Extracting and Visualising Stock Data.ipynb](./Extracting%20and%20Visualising%20Stock%20Data.ipynb) — Jupyter Notebook (code + markdown)
- [Extracting and Visualising Stock Data.html](./Extracting%20and%20Visualising%20Stock%20Data.html) — HTML version with plots and outputs

## Description
The notebook extracts stock data for Tesla and GameStop, scrapes quarterly revenue data, and visualizes both datasets using Plotly.
